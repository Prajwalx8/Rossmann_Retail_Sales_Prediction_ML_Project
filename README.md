# 📊 Rossmann Store Sales Prediction using Machine Learning

## 📌 Project Overview

This project focuses on forecasting **daily sales for Rossmann drug stores** using Machine Learning techniques. The objective is to help retailers make data-driven decisions for inventory planning, workforce management, promotional campaigns, and demand forecasting.

The project follows a complete **end-to-end Machine Learning pipeline**, including data preprocessing, exploratory data analysis (EDA), feature engineering, model building, hyperparameter tuning, and performance evaluation.

---

## 🎯 Business Problem

Rossmann operates over **3,000 drug stores across seven European countries**. Store managers are required to predict daily sales up to six weeks in advance. Sales are influenced by several factors, including:

* Promotions
* Competition
* School & State Holidays
* Seasonality
* Customer Footfall
* Store Type
* Assortment

The goal is to build a predictive model capable of accurately forecasting daily sales.

---

## 📂 Dataset

The project uses two datasets:

* **Rossmann Stores Data** – Historical daily sales records
* **Store Data** – Store-specific information such as competition distance, assortment, promotions, and store type

Total Records:

* **1,017,209 Sales Records**
* **1,115 Stores**

## 📈 Model Performance

| Model                   | Test R² Score |
| ----------------------- | ------------- |
| Linear Regression       | **0.7949**    |
| Random Forest Regressor | **0.8264** ✅  |
| LARS Lasso Regression   | **0.7949**    |

### Final Model

✅ **Random Forest Regressor** was selected as the best-performing model.

## 💡 Key Business Insights

* Customer count is the strongest predictor of sales.
* Promotions significantly increase store sales.
* Store Type A generates the highest sales.
* Store Type D has the highest average customer spending.
* School Holidays positively impact sales.
* Christmas season records peak sales.
* Promo2 shows limited impact on sales performance.
